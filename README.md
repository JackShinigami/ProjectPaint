# ProjectPaint
 Second Project of Windows Programing Course


## Các phím tắt
- 'Ctrl + N' : Tạo một layer mới
- 'Ctrl + O' : Mở một file ảnh
- 'Ctrl + S' : Lưu bản vẽ dưới dạng file ảnh
- 'Ctrl + Z' : Undo
- 'Ctrl + Y' : Redo
- 'Ctrl + C' : Copy
- 'Ctrl + V' : Paste
- 'Ctrl + X' : Cut
- 'Ctrl + E' : Chuyển đổi giữa Edit Mode và Draw Mode
